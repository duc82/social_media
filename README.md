# Social Media

> A simple social media application that allows users to share their thoughts and photos with the world. It aims to provide a platform for users to connect with others and share their experiences.

## Features

- User authentication (Sign up, Login, Logout)
- Profile creation and editing
- Post creation, liking, and commenting
- Newsfeed with posts from friends
- Follow and unfollow users
- Real-time chat with friends
- Video calling functionality
- Notifications for new messages and friend
- Groups for sharing posts with multiple users
- Admin area for managing users and posts
- Dark/light mode toggle

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/duc82/social-media.git
   ```

2. Install the dependencies:

   ```bash
   # client
    cd client
    npm install

   # server
    cd server
    npm install
   ```

## Usage

### Environment Variables

Rename the `.env.example` files in the `client` and `server` directories to `.env` and update the variables with your own values.

### Running the Application

```bash
# client (:3000)
cd client
npm run dev

# server (:5000)
cd server
npm run start:dev
```
